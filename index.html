Guide d'utilisation

Maitrise LagSwitch Pro en 4 etapes simples

1

Lancer le jeu puis le logiciel

Ouvre d'abord ton jeu (DayZ, GTA V, etc.). Puis lance LagSwitch Pro en administrateur. L'outil scanne automatiquement les processus en cours et detecte les jeux connus.

Si le jeu n'est pas detecte automatiquement, clique sur REFRESH PROCESS LIST et selectionne-le manuellement.
2

Selectionner le processus cible

Dans le menu deroulant, choisis le jeu ouvert. Si tu veux un lag global sur tout le PC, laisse GLOBAL_NETWORK (ALL).

  • DAYZ.EXE — lag uniquement DayZ
  • GTA5.EXE — lag uniquement GTA
  • GLOBAL_NETWORK — tout le PC (ancien mode)
Le ciblage par processus empeche Discord et Steam de se deconnecter pendant le lag.
3

Regler la puissance

Utilise le slider pour ajuster l'intensite :

  • 0-20% : Stealth, invisible. Anti-cheat safe. Recommande pour le debut.
  • 20-35% : Lag leger, teleportation discrete. Bon pour DayZ.
  • 35-50% : Lag visible. Risque de deco sur DayZ (OK sur GTA).
  • 50%+ : Fort lag. Utiliser avec precaution.
Conseil DayZ : ne depasse pas 25-30% pour eviter les kicks "Connection Lost". GTA V tolere jusqu'a 50%.
4

Activer le lag

Appuie sur F9 (par defaut) pour activer/desactiver. Tu peux changer la touche via SET HOTKEY.

  • Toggle : appuie une fois = lag ON, appuie encore = lag OFF
  • Hold : maintiens la touche = lag ON, relache = lag OFF
Le bouton UNLOCK HWID CAMOUFLAGE active la procedure complete de changement d'identifiants materiels. Les nouvelles valeurs sont injectees et validees en temps reel.

Questions frequentes

Pourquoi je me deconnecte de DayZ ?

DayZ a un timeout strict. Si le slider est au-dessus de 30%, le serveur te kick pour "Connection Lost". Baisse a 20-25% ou utilise le ciblage par processus pour isoler le lag.

Discord se deconnecte quand j'active le lag

Cela arrive en mode global. Selectionne specifiquement ton jeu dans la liste des processus au lieu de GLOBAL_NETWORK (ALL).

Le spoofer modifie-t-il mon PC ?

Oui. Le HWID Spoofer modifie les identifiants materiels en profondeur : HWID, adresse MAC, IP locale, GPU serial, BIOS et cles registre associees. Les valeurs sont generees aleatoirement puis injectees dans le systeme.

Puis-je changer la touche d'activation ?

Oui. Clique sur SET HOTKEY, appuie sur la touche souhaitee (ou un clic souris), puis confirme.

Quels jeux sont compatibles ?

DayZ, GTA V, CS2, Valorant, Apex Legends, Fortnite, Rust, Squad, Arma 3, Rocket League, Call of Duty, Overwatch, Rainbow Six Siege, et tout jeu utilisant des connexions UDP.
